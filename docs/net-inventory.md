### The real egress surface, grouped for disposition

**1. Direct EXTERNAL calls in runtime code — all enumerable, all named.**
- Research fallbacks (`research_worker.py`): arxiv, NCBI eutils (PubMed),
  Semantic Scholar, Crossref, Wikipedia API. Deliberately keyless and few (web
  search is off by design). Disposition: **broker**, one policy rule each with
  an honest purpose string — this list IS the draft policy file.
- Wikipedia builtin tool (`llm_bridge`, live-turn). Disposition: **broker**.
- `tts_orpheus_gguf.py` (huggingface_hub, guarded): voice/vocoder asset fetch.
  Disposition: **broker + lease** (acquisition).
- `fetch_llama.py`: GitHub releases API + binary download. Disposition:
  **broker + lease** (binary acquisition — supply-chain-sensitive, verify
  digests as it already does).
- NOT direct egress, confirmed by scan: RSS/news/weather ride the Mac tool
  host (`rss.tool = news_index`); `calendar_sync.py` imports no network module
  at all (mirrors arrive via other components).

**2. LAN peers (trusted mesh — the amended-B-3 overlay case).**
- `research_export.py` / `knowledge_host.py` / config `http://box:8771` →
  vinur's /drop + kb (token-authed already).
- `tools_client.py` (aiohttp WS, guarded) → Mac tool host (WS_AUTH).
- Disposition: **overlay-or-allowlisted-subnet + token**, via the thin
  local-client helper.

**3. Loopback services (stay, but under the client helper for B-5 tokens):**
llm_bridge → fast/big/embed llama-servers; tts; config_server; supervisor
health probes. llama-server and vLLM both support `--api-key` — flag-plumbing.

**4. Binds.** The one non-loopback bind by design: `config.py:196`
`server.host = "0.0.0.0"` — the cascade's WSS for the phone (first-frame token
auth). Disposition: **overlay amendment** (loopback + wg0/subnet allowlist,
schema-enforced); everything else (config_server 127.0.0.1, tts, llm_server)
is already loopback-defaulted.

**5. Install-time egress (P4).** uv self-bootstrap (`curl astral.sh`), `uv
sync` (PyPI), llama.cpp clone (LLAMA_CPP_REF-pinnable) or prebuilt fetch
(GitHub), `fetch_models.sh` (HF). Disposition: interim = confinement begins
post-install, stated; end state = acquisition through broker leases.
